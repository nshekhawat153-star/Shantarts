# Shanta Arts 

> *Handcrafted Rajasthani Art for Timeless Modern Interiors.*

Shanta Arts is a premium, minimal, luxury heritage art brand and editorial interior design studio. Originally founded in the 1990s as a family-run business exporting traditional Rajasthani paintings globally, Shanta Arts has been revived to seamlessly combine authentic Indian craftsmanship with modern luxury interior aesthetics.

The brand collaborates directly with master artisans from Rajasthan, working alongside interior designers, architects, luxury homeowners, and high-end hospitality developers (hotels, resorts, and luxury villas) to create bespoke, heirloom-quality artworks.

---

## 🎨 Visual Identity & Style Direction

The design of the digital platform is heavily inspired by editorial publications and high-end luxury hospitality brands like **Aman Resorts**, **Architectural Digest**, and **Nilaya by Asian Paints**.

* **Palette**: Warm Ivory, Sandstone Beige, Antique Gold accents, and Matte Charcoal.
* **Typography**: Elegant `Cormorant Garamond` serif for editorial headings, balanced with clean, modern `Inter` sans-serif for reading comfort.
* **Layout**: Spacious, asymmetric grids, extensive whitespace, and beautiful custom interactive timelines.

---

## ✨ Features & Interactive Elements

1. **Editorial Hero & Narrative**: A stunning, full-height landing experience leading into an emotional brand story of revival and artisan preservation.
2. **Asymmetric Collection Showcase**: Clean category grids featuring:
   * Pichwai Paintings
   * Rajasthani Miniature Art
   * Custom Wall Murals
   * Heritage Canvas Art
   * Palace-Inspired Masterpieces
   * Contemporary Fusion Art
3. **Trade Partnerships Portal**: A dedicated showcase outlining commission opportunities, custom configurations, and margin scales for architects and interior design professionals.
4. **Interactive Creation Timeline**: A visual, step-by-step progress explorer displaying the complete lifecycle of a custom heritage art commission from consultation to final framing.
5. **Close-up Texture Lightbox**: An interactive masonry gallery with built-in close-up zoom states to inspect the fine brushwork and natural stone pigments of each masterpiece.
6. **Luxury Consultation Form**: A seamless, modern form with dynamic budget selectors and file drop-zones designed to capture precise requirements for custom high-end art installations.
7. **SEO-Optimized & Structured FAQ**: Contains an interactive, beautifully-animated accordion handling key customer queries, integrated with valid **JSON-LD Schema Markup** for rich search results.

---

## 🛠️ Technology Stack

* **Structure**: Semantic HTML5
* **Styling**: Vanilla CSS3 (Custom design system, fluid type scales, responsive units, and cubic-bezier micro-animations)
* **Interactivity**: Vanilla ES6+ JavaScript (Scroll-triggered animations, interactive slider, dynamic accordion heights, timeline progress indicators, and custom modal handling)
* **SEO & Metadata**: Optimized heading structures (strictly one single `<h1>`), highly-descriptive image `alt` text mapping targeted heritage art keywords, and valid structured JSON-LD data.

## 🚀 How to Run Locally

### 1. Direct File Load
Open your browser and drag the `index.html` file directly into any tab, or double-click the file inside the project directory.

### 2. Local Development Server
For the best experience with full asset loading, spin up a local server using `npx`:
```bash
npx http-server
```
Once the server is running, navigate to:
```
http://localhost:8080
```
